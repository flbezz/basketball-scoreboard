# 🏀 Basketball Match Scoreboard

high-contrast digital basketball scoreboard. This project focuses on responsive DOM manipulation and layout design to track real-time match scoring for Home and Guest teams.

## ✨ Features
*   **Dual Team Tracking:** Independent scoring controls for both **Home** and **Guest** teams.
*   **Point Increments:** Quick-action buttons to seamlessly add `+1`, `+2`, or `+3` points instantly.
*   **Sleek Digital Display:** Utilizes high-contrast, glowing red digital typography against a deep dark background for an authentic scoreboard feel.

## 🛠️ Tech Stack & Design Choices
*   **HTML5:** Structured semantic markup using sections and semantic naming conventions.
*   **CSS3:** Styled with custom layout structuring, leveraging `flexbox` for alignment and specific color accents.
    *   **Typography:** Featuring the `Orbitron` font for that classic digital stadium clock aesthetic, paired with `Montserrat` for clean headers.
*   **Vanilla JavaScript:** Powered by crisp, logic-first DOM manipulation without relying on external frameworks.
